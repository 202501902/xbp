Arduino


Arduinoの基本

Arduinoとは入出力ポートがあるマイコンボードである。
プログラムを書くことで、電子工作といわれることが簡単にできる。
例：センサーの値を読み込む。モーター動かす。LEDを光らせる。

ArduinoとPCはUSBで接続する。(付属のケーブルはtypeA)
Arduino Unoというボード上でプログラムを書く。


Lチカ

LEDを光らせる



    void setup() {
       pinMode(13, OUTPUT);
    }

    void loop() {
      digitalWrite(13, HIGH);
      delay(500);
      digitalWrite(13, LOW);
      delay(500);
    }

◎void：実行される回数
setup：最初に一回だけ実行されるプログラム
loop：ずっと繰り返し実行されるプログラム

;(セミコロン)：基本的に命令文の最後につける

pinMode：指定したピンを入力用に使うのか、出力用に使うのかを設定する
LEDを点灯させるために13番ピンから電気を流したい（出力したい）→pinMode(13,OUTPUT);



LEDを点滅させる


0.3秒ごとの点滅を10回繰り返し、その後２秒ごとの点滅を３回繰り返す、というプログラム。
最後まで実行されたら、また同じことの繰り返し。



    void setup() {
      // put your setup code here, to run once:
      pinMode(13,OUTPUT);
    }

    void loop() {
      // put your main code here, to run repeatedly:
      for(int i=0; i < 10; i++){
        digitalWrite(13,HIGH);
        delay(300);
        digitalWrite(13,LOW);
        delay(300);
      }
  
      for(int i=0; i < 3; i++){
        digitalWrite(13,HIGH);
        delay(2000);
        digitalWrite(13,LOW);
        delay(2000);
      }

    }

for文とif文

for文



    for (初期化式; 条件式; 変化式){
      // 繰り返しの中で実行される処理
    }

使用例



    for(int i=0; i < 3; i++){
      // 繰り返しの中で実行される処理
    }

iという変数で繰り返しを判断する
iは0から始まる
iが３未満の間は繰り返す（0から始まるので、３回の繰り返しになることに注意）
i++はiが１ずつ足される（数を１，２，３と数えている感じ）


if文



    if (条件式){
    　処理内容1 　//条件式を満たす場合にのみ実行する
    }
    else{
    　処理内容2 　//条件式を満たさない場合にのみ実行される
    }
