Flaskの基本

PythonでWebサイトやAPIを開発するための小規模フレームワークである。
必要最低限の機能だけを備えており、開発者が自由に機能を追加しながらアプリを作れる。
作成例：Webサイト、ブログ、管理画面、AIアプリのWeb UI、社内ツール

今日の運勢アプリ


python


from flask import Flask, render_template, request
import random

app = Flask(__name__)

# 運勢のリスト
fortunes = [
    "大吉！！！",
    "吉！！",
    "中吉！！",
    "小吉！",
    "末吉！",
    "凶..."
]

@app.route("/", methods=["GET", "POST"])
def index():

    result = ""

    # ボタンが押された時
    if request.method == "POST":

        # ランダムで1つ選ぶ
        result = random.choice(fortunes)

    return render_template("index.html", result=result)

if __name__ == "__main__":
    app.run(debug=True)
